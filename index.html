Sobre

Olá, meu nome é William Kotesky e sou um desenvolvedor web frontend com fortes aspirações de me tornar fullstack em um futuro próximo. Minha história é um pouco incomum, pois comecei minha jornada estudando Engenharia Química na USP, mas percebi que isso não me fazia feliz. Em busca de novos conhecimentos para melhorar meu currículo, acabei descobrindo Python e sua lógica de programação, o que me mostrou o caminho para um futuro promissor. Hoje, com o JavaScript, tenho a capacidade de desenvolver páginas web e busco constantemente melhorar minhas habilidades a cada dia.

Conhecimentos

HTML
Domínio dos headings, tags normais e semânticas, atributos das tags e uso completo dos forms. Regras semânticas, estruturas próprias do HTML como o próprio form, tabelas, media, audio, video.

CSS
Domínio dos diferentes seletores, especificidade, atributos de um elemento em específico como suas dimensões, borda, padding, margin, bem como as diferentes formas de se posicionar um elemento na tela com os diversos tipos de display(relative/absolute/flexbox/grid) além de ferramentas que auxiliam nisso, como overflow, z-index, float, math functions. Domínio geral sob a formatação de textos, tabelas, forms, botões, transições e animações bem como o uso de variáveis e construção de componentes como dropbox, cards, carousel e etc. Uso da responsividade com media queries, optando pelo mobile first.

Javascript
Tipos de variáveis e de declaração de variáveis, template strings, operadores matemáticos e lógicos objeto math, objeto date, if/else, operação ternária, switch/case, atribuição via desestruturação (arrays e objetos), for, for in, for of, while, do while, break/continue, try-catch-throw, try-catch-finally, setInterval, setTimeout, strings e seus métodos. Propriedades das funções (argumento/parâmetro, return, escopo, closures) e os diversos tipos de funções. Arrays e seus métodos, incluindo filter, map, reduce, forEach. Objetos e prototypes, getter/setter, métodos, heranças e polimorfismo e orientação à objetos. Assíncrono, promises e seus diferentes métodos, async/await, fetch API e axios. Javascript Tooling - babel e webpack, ES6 modules - import/export.

ReactJS
Estrutura geral do React - components e props, states e como o framework lida com a renderização de elementos na página, JSX. Hooks: useState, useEffect, useRef, useId, useContext.

Bootstrap
Breakpoints, containers, grid, gutters e todos os outros elementos de posicionamento. Customização completa de forms e seus tipos, incluindo validação. Todos os componentes que o Bootstrap oferece, bem como todos os helpers e utilities.

NodeJs
Node modules básico, express básico e conexão simples com mongoDB. NPM - instalação de pacotes, instalação de versões específicas de um pacote, dependences/dev-dependences, mover os pacotes entre essas dependencias, atualizar pacotes de versões major, patch e minor, listar pacotes instalados, desinstalar items e mostrar aqules que precisam de atualização.

Typescript
Domínio dos diversos tipos que essa linguagem oferece: Any, void, object, array, tuple, null, undefined, never, enum, unknown, união e interseção de tipos, funções e classes como tipos, type alias, type assertions, estrutura de tipos. Também toda a parte de classes, modificadores public, private, protected, heranças, super, getters e setters, atributos e métodos estáticos, construtor privado/singleton, classes/métodos/atributos abstratos, associação/agregação/composição entre classes e interfaces.

Outros
O básico de Python: condicional, laços, funções, sequências, strings. Excel intermediário. Inglês Avançado: Conversação, leitura e escrita em inglês, familiaridade com vocabulário utilizado na programação. React router dom, webpack, eslint e um pequeno contato com mongoDB e Express. Photoshop básico.

Projetos

NFL INFO BR
Site | Repositório
Um projeto feito com React.Js que simula um site sobre o mundo do futebol americano da NFL. Existem nove páginas, contando com endereços dinâmicos que exibem informações de itens clicados, como notícias e detalhes sobre os times favoritos. Há como adicionar, editar e excluir notícias e times, usando o Local Storage do Javascript, funcionando como um banco de dados. Também há ligações com API's que buscam informações como o calendário da semana e estatísticas detalhadas dos times adicionados pelo usuário.
